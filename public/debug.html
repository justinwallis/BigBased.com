Debug Tools

This page provides access to various debugging tools for the Big Based website.

Static Route Debug

These pages are served by route handlers with no JavaScript:

  • Static Route Debug Home
  • Favicon Debug (Static)
  • Preloader Debug (Static)

Static HTML Debug

These pages are completely static HTML files:

  • Debug Home (this page)
  • Minimal Debug
  • Static Debug

Next.js Debug

These pages use Next.js but might not work if there are JavaScript errors:

  • Debug Home
  • Debug Home (No JS)
  • Preloader Debug
  • Favicon Debug

Preloader Issues

If you're experiencing issues with the preloader, try these solutions:

  • Disable the preloader
  • View preloader debugging instructions

Troubleshooting

If you're having trouble with the debug pages, try the following:

  1. Start with the static HTML pages (like this one)
  2. Then try the static route debug pages
  3. Finally, try the Next.js debug pages with the nojs parameter

If none of these work, there might be a more serious issue with the site.
